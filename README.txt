Description
===========
This module adds the necessary JS to the footer of your website for comScore direct implementation.
http://direct.comscore.com/


Overview
A simple module to add in the JS for the comScore Direct tag to your Drupal site
http://direct.comscore.com/
Features
Includes an admin interface to add
comScore ID (required)
comScore Genre of content (optional)
comScore Package (optional)
comScore Client Segment ID (optional)
Requirements
No dependencies
